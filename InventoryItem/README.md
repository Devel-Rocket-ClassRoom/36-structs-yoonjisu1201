# 인벤토리 아이템 관리

게임 인벤토리 아이템을 나타내는 `InventoryItem` 구조체를 만들고, 아이템 배열을 활용하여 다양한 정보를 
계산하는 프로그램을 작성하세요.

**요구사항:**

1. `InventoryItem` 구조체를 정의하세요.
   - `Name` (string), `Weight` (double), `Price` (int) 필드
   - 세 값을 받아 초기화하는 생성자

2. 5개의 아이템 배열을 생성하세요.
   - 검 (3.5kg, 1500골드)
   - 방패 (5.0kg, 1200골드)
   - 포션 (0.3kg, 50골드)
   - 활 (1.8kg, 1300골드)
   - 투구 (2.0kg, 800골드)

3. 다음 정보를 계산하여 출력하세요.
   - 전체 아이템 목록 (이름, 무게, 가격)
   - 전체 무게 합계
   - 전체 가격 합계
   - 평균 가격
   - 가장 비싼 아이템 이름
   - 가장 가벼운 아이템 이름

## 예상 실행 결과

```
=== 인벤토리 목록 ===
검 - 무게: 3.5kg, 가격: 1500골드
방패 - 무게: 5kg, 가격: 1200골드
포션 - 무게: 0.3kg, 가격: 50골드
활 - 무게: 1.8kg, 가격: 1300골드
투구 - 무게: 2kg, 가격: 800골드

=== 인벤토리 통계 ===
전체 무게: 12.6kg
전체 가격: 4850골드
평균 가격: 970골드
가장 비싼 아이템: 검
가장 가벼운 아이템: 포션
```
